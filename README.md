Comma 是一款帮助您学习英语的桌面应用。

[Comma下载、使用简介](https://peng-creator.github.io/comma-web/comma/)

## 开发者
### 拉取代码
```
git clone https://github.com/peng-creator/Comma.git
```
### 安装依赖
```
cd Comma && yarn
```

### 开发调试
```
yarn start
```

### 打包调试
```
yarn package:debug
```

### 打包
```
yarn package
```

### 使用安装包
打包成功后，在项目内找到release目录，双击 .exe （windows）或 .dmg（MacOS）引导安装。
